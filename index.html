Australian Communication Theory Workshop 2026

Welcome to AusCTW2026. More coming soon. This is using a conference website template: It’s got a page for registration, where you can embed a Google Form or some other way for participants to register. It’s got a program page to list the schedule for the conference. It’s got a venue page to add a bunch of instructions how to get to the conference. It’s got a CFP to put up the Call for Papers and other promotional materials for folks to download.

I cloned all the files from this GitHub repo here. This template was created from a website MathConnections 2019, in case you want to see an example of this template being used in the wild.
